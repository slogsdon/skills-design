01 / The pattern

Software interfaces went UX → DX → AX.

Each shift redesigned the same surfaces — interfaces, APIs, documentation — around a different reader. The latest reader is an agent operating autonomously between sessions.

02 / The symptom

When sessions drift, the prompt is rarely the problem.

I refined prompts for six months. The post-generation corrections list looked nearly identical to month two. The thing I kept adjusting wasn't the problem.

§ The AX Shift — three observations 04 / 05

03 / The fix

Write for the agent reading it cold.

Stop assuming continuity that doesn't exist. The agent doesn't remember last week. Inference at the edges of the session is exactly what causes the drift you've been blaming on prompts.
