# skills-design

Brand-agnostic design system and 22 artifact generators for Claude Code, plus an HTML-to-PNG export utility. 23 skills total.

Install via the [slogsdon-claude-code-config marketplace](https://github.com/slogsdon/claude-code-config).

## Skills

- **design-blog-hero** — Blog hero image generator
- **design-business-card** — Business card design
- **design-carousel-slide** — Carousel slide generator
- **design-instagram-post** — Instagram post design
- **design-landing-page** — Landing page design
- **design-link-in-bio** — Link-in-bio page design
- **design-linkedin-post** — LinkedIn post design
- **design-newsletter-header** — Newsletter header design
- **design-obs-alert-overlay** — OBS alert overlay design
- **design-obs-scene-pack** — OBS scene pack design
- **design-plan** — Design planning and briefing
- **design-podcast-cover** — Podcast cover art design
- **design-quote-card** — Quote card design
- **design-speaker-bio-card** — Speaker bio card design
- **design-stream-overlay** — Stream overlay design
- **design-system** — Design system tokens and components
- **design-talk-slide** — Talk/presentation slide design
- **design-twitch-panels** — Twitch panel design
- **design-twitter-card** — Twitter card design
- **design-ui-components** — UI component design
- **design-youtube-channel-art** — YouTube channel art design
- **design-youtube-thumbnail** — YouTube thumbnail design
- **screenshot-html** — Export HTML artifacts as PNG screenshots
